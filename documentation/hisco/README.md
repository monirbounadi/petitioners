# SwedPop HISCO documentation

This publicly available file documents the occupational-classification principles used in release v0.1:

- `SwedPop-occupation-coding-documentation.pdf`: Annika Westberg, *Principles of Coding Swedish Historic Occupations*, CEDAR/SwedPop, version 19 August 2022.

The matching used a SwedPop occupational-title and HISCO code-list workbook supplied to the authors. Because that workbook is not known to be freely redistributable, it is not included in the public release. The PDF is included unchanged and remains attributable to SwedPop/CEDAR and its named author.
